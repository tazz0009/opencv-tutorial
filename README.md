# OpenCV Python Tutorial

## chap00

- 가상환경설치

```
python3 -m venv opencv-env
opencv-env\Scripts\activate.bat
```

- 아나콘다 가상환경설치

```
conda create -n opencv-env
conda info --envs
conda activate opencv-env
conda deactivate
```

- 필수 라이브러리 설치

```
pip3 install -r requirements.txt
```
